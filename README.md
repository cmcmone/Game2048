# 2048 GAME

An number puzzle game made with HTML/CSS and JavaScript.

## How to Play:
1. Use the `git` command to clone this repository:
```
git clone https://github.com/cmcmone/Game2048.git
```

2. Use a browser to open the `index.html` file
